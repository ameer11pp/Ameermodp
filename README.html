كيفية استخدام عكسلي

1

تحميل الملف

قم بتحديد ملف LIB أو SO من جهازك. يمكنك سحبه وأفلاته في المنطقة المخصصة أو النقر لاختيار الملف.

2

بدء عملية الفك

انقر على زر "بدء عملية فك التشفير" لبدء تحليل الملف وفك تشفير الشيفرة البرمجية.

3

تحليل النتائج

بعد اكتمال العملية، ستظهر الشيفرة البرمجية المفكوكة في منطقة النتائج. يمكنك تصفحها وتحليلها.

4

تحميل النتائج

قم بتنزيل الشيفرة المصدرية كملف نصي لاستخدامها في مشاريعك أو لتحليل أعمق باستخدام أدوات أخرى.
